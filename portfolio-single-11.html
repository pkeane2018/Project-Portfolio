Google Books Search Supreme

Project Details

  • Tools Used: React, MongoDB, Mongoose, Express, Nodejs, Axios
  • APIs Used:
    Google Books
  • repository
  • site

Users can use this site to search the Google Books API for information on books. After entering the title of the book in the search bar and pressing Submit, the results of the search will display on the page. In each results entry, clicking the "View" button will open a link to a the Google Books page for that particular book. Clicking the "Save" button will save the entry to a database. Clicking on "Saved" in the navbar will bring the user to another page which will display all books they previously saved. In each of the entries in this page, clicking the "View" will again open up a new tab to the Google Books page for that book, whereas clicking the "Delete" button will delete the book from the database and update the page to display the books still in the database.
