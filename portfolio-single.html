Storage Supreme

Project Details

  • Tools Used: MySQL, JawsDB, Express, Nodejs
  • repository
  • site

This site is a supply chain management system that allows users to submit orders of supplies for construction projects and update and keep track of the status of those orders. The homepage links to three other pages - the "New Request" page where users can request supplies for new projects, the "Procurement Portal" page where users can action the requests and update the delivery dates and statuses, and the "Review Request Status" page where users can review all requests.
