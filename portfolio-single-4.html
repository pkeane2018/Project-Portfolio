Exoplanet Finder

Project Details

  • Tools Used: jQuery, AJAX, Firebase
  • APIs Used: NASA Exoplanet API
  • Other Contributors: Spencer Caretta, Eric King
  • repository
  • site

Users can search for information on known exoplanets. The program retrieves data from the NASA Exoplanet API and allows users to narrow their searches by certain parameters. Results are displayed in a table. Users can then click on the names of an individual planet to open a seperate tab with more information on it.
