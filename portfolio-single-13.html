Vox News Scraper

Project Details

  • Tools Used: Express, Express-Handlebars, Nodejs, Axios, Cheerio, Moongoose, MongoDB
  • repository
  • site

This website scrapes data from the Vox news homepage on breaking news stories, and displays this information on the page. This includes the headine, a summary of the story, and a link to the story webpage. Every time the page is opened, it scrapes data from the Vox homepage again and displays the updated information to the screen. The user can also type his or her username and a comment in their respective boxes. Clicking the "Add comment" button saves the username and comment to a database and then displays the new comment and username in a box beneath the assoicated story. Clicking the "Delete Comment" will delete the comment from the database and remove it from the page. All of the stories, comments, and usernames added to the page are saved and loaded to the page every time it is opened.
